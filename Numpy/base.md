# Numpy
## Создание массивов
- np.array -  создать массив
- np.shape - количество элементов внутри массива и внутри них
- np.ndim - вложенность элементов
- np.zeros(n) - создание массива из n кол-ва нулей
- np.zeros((n, m)) - создание массива из под массивов n, с m нулей внутри каждого
- np.ones(n) - создание массива из n кол-ва единиц
- np.ones((n, m)) - создание массива из под массивов n, с m единиц внутри каждого
- np.full(n, m) - создание массива из n кол-ва m элементов
- np.full((n, l), m) - создание массива из под массивов n, с l кол-вом m элементов внутри каждого
- np.arange(n) - создание списка из n элементов начиная с 0
  ## Работа с типами
- np.dtype - тип элементов в массиве
- m.astype(np.t) - возвращает в новый контейнер значения старого контейнера m переводя значения в тип данных t. При переводе флоат в инт, он обрезает часть после точки а не округляет. Также можно писать так m.astype(r.dtype) и он приведет тип массива m к типу массива r
## Срезы и индексинг
- m2 = m1[3:5].copy() - скопирует срез m1 в отдельный массив m2, который не будет менять первоначальный массив
- m[[0,2,4]] - фэнси индексинг, в таком виде из массива вытянутся 1, 3 и 5 строки, причем можно записывать элементы в свободном порядке
## Преобразование массивов
- m1.reshape((m, e)) - Преобразует массив m1 в двумерный массив с количеством строк m и количеством элементов e (m*e должно равняться количество элеменов в изначальном массиве). Если нужен трехмерный массив, то в кортеже передаем (n, m, e) где n будет количеством двумерных матриц
- m.T или m.transpose() - Транспонирование двумерного массива
## Арифметические функции 
- np.sqrt(m) - извлечение корня из всех элементов массива m
- np.abs(m) - модули из всех элементов массива m
- np.round(m) - округление всех элементов массива m
- np.ceil(m) - округление вверх всех элементов массива m
- np.floor(m) - округление вниз всех элементов массива m
- np.nan - отсутствие значений как None в питоне или #Н/Д в экселе. ставить вместо элементов в массиве
- np.isnan(m) - выдает булевый массив где все True будут обозначать отсутствие значений
- np....(m, out=n) - параметр out говорит функциям, которые работают поэлементно в массиве m вернуть значения в массив n. Имя входного и выходного массива могут быть одинаковыми
- np.maximum(m1, m2) - в отдельный массив складывает максимальный элемент каждого массива
- np.minimum(m1, m2) - в отдельный массив складывает минимальный элемент каждого массива
- np.add(m1, m2) - сложение элементов двух массивов
- np.subtract(m1, m2) - вычитание элементов второго массива из первого
- np.multiply(m1, m2) - перемножение элементов двух массивов
- np.divide(m1, m2) - деление элементов двух массивов m1\m2
- np.greater(m1, m2) - Отдает булевый массив, где будут показаны результаты сравнения массива m1 относительно m2, помечая как True элементы больше. Аналог функции m1 > m2
- np.greater_equal(m1, m2) - Отдает булевый массив, где будут показаны результаты сравнения массива m1 относительно m2, помечая как True элементы больше или равны. Аналог функции m1 >= m2
- np.less(m1, m2) - Отдает булевый массив, где будут показаны результаты сравнения массива m1 относительно m2, помечая как True элементы меньше. Аналог функции m1 < m2
- np.less_equal(m1, m2) - Отдает булевый массив, где будут показаны результаты сравнения массива m1 относительно m2, помечая как True элементы меньше или равны. Аналог функции m1 <= m2
- np.equal(m1, m2) - Отдает булевый массив, где будут показаны результаты сравнения массива m1 относительно m2, помечая как True элементы равны. Аналог функции m1 == m2
- np.not_equal(m1, m2) - Отдает булевый массив, где будут показаны результаты сравнения массива m1 относительно m2, помечая как True элементы не равны. Аналог функции m1 != m2
- np.isinf(m) - выводит булевый список отмечающий каждую бесконечность, возникающая в основном при делении на 0
