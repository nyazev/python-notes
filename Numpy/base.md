# Numpy
- np.array -  создать массив
- np.shape - количество элементов внутри массива и внутри них
- np.ndim - вложенность элементов
- np.dtype - тип элементов в массиве
- np.zeros(n) - создание массива из n кол-ва нулей
- np.zeros((n, m)) - создание массива из под массивов n, с m нулей внутри каждого
- np.ones(n) - создание массива из n кол-ва единиц
- np.ones((n, m)) - создание массива из под массивов n, с m единиц внутри каждого
- np.full(n, m) - создание массива из n кол-ва m элементов
- np.full((n, l), m) - создание массива из под массивов n, с l кол-вом m элементов внутри каждого
- np.arange(n) - создание списка из n элементов начиная с 0
